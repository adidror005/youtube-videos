Modern Portfolio Theory (MPT) Overview¶

Intro¶
  • Question would you prefer portfolio A or portfolio B?
    • Modern Portfolio Theory provides an answer to this
Overview¶
  • Harry Markowitz: Developed by economist Harry Markowitz in the 1950s for which he received Nobel Prize
  • Risk-return tradeoff: Focuses on optimizing portfolio allocation by balancing risk and return.
  • Diversification: The main idea behind MPT is diversification: by combining different assets, investors can reduce overall portfolio risk without sacrificing expected returns.
Risk-Averse Investor's Preference in MPT¶

Assumptions:

  • Risk Aversion: Investors prefer less risk for the same or higher returns.
  • Return and Risk: Let $\mu_A$ and $\sigma_A$ be the expected return and risk (standard deviation) of Portfolio A, and $\mu_B$ and $\sigma_B$ for Portfolio B. For a risk-averse investor, a higher return must not come with higher risk.
  • Preference Condition:
  • Suppose we have 2 portfolios A and B
  • $\mu_A$ and $\mu_B$ are the expected returns for portfolios A and B respectively
  • $\sigma_A$ and $\sigma_B$ are their expected risks (standard deviations) $$ \mu_A \geq \mu_B \iff \sigma_A \geq \sigma_B $$

This simple condition emphasizes that risk-averse investors are unwilling to trade off higher returns for higher risk, reinforcing the principles of portfolio optimization under MPT.

¶

Key concepts in MPT include:

  1. Risk and Return Trade-off:

    • MPT quantifies risk using the standard deviation of asset returns
    • Asserts that investors should seek portfolios that maximize expected return for a given level of risk or, conversely, minimize risk for a given level of return.

  2. Covariance/Correlation: The relationship between the returns of different assets. Diversifying across assets with low or negative correlation can reduce overall portfolio risk.

  3. Portfolio Diversification: By including a variety of uncorrelated or negatively correlated assets, investors can minimize the portfolio's overall risk. According to MPT, a diversified portfolio reduces unsystematic risk (risk specific to individual investments) while systematic risk (market-wide risk) remains.

  4. Efficient Frontier: MPT suggests that, for a given level of risk, there exists an optimal set of portfolios that offer the highest expected return. These portfolios make up the efficient frontier, a curve that represents the most efficient risk-return combinations.

Mathematical Formulation¶

To construct the efficient frontier in Markowitz's Modern Portfolio Theory (MPT), we typically use two equivalent optimization problem formulations:

  1. Minimizing Portfolio Variance for a Given Expected Return
  2. Maximizing Expected Return for a Given Level of Risk

Each point on the efficient frontier corresponds to a portfolio that has an optimal trade-off between risk and return.

Let’s define some key terms for these formulations:

  • $S_1,\dots,S_n$: n assets
  • $\mathbf{w} = [w_1, w_2, \dots, w_n]^\top$: The vector of portfolio weights for $n$ assets.
  • $\mathbf{\mu} = [\mu_1, \mu_2, \dots, \mu_n]^\top$: The vector of expected returns for each asset.
  • $\Sigma$: The $n \times n$ covariance matrix of asset returns.
  • $R_{\text{target}}$: The target expected return.
  • $\sigma^2$: The portfolio variance (a measure of risk), calculated as $\mathbf{w}^\top \Sigma \mathbf{w}$.

So portfolio $\Pi$ is defined as weighted combiniation of $n$ assets: $$\Pi=\displaystyle\sum_{i=1}^{n}W_iA_i$$ So, $$\mathbf{E}\left[\Pi\right]=\displaystyle\sum_{i=1}^{n}W_i\mu_i=\mathbf{w}^T\mathbf{\mu}$$ and $$\mathbf{\text{Var}}\left[\Pi\right]=\displaystyle\sum_{i=1}^{n}W_i^2=\frac{1}{n}\mathbf{w}^T\mathbf{\mu}$$

Reminder: Mean and Variance of Sum of Random Variables¶

The expectation $\mathbf{E}$ of weighted sum of $n$ random variables $X_1,\dots,X_n$ is $X=\sum_{i=1}^nw_iX_i$ is:

$$\mathbb{E}[X] = \sum_{i=1}^n w_i\mathbb{E}[X_i]$$

The variance $\mathbf{\text{Var}}$ is:

$$\mathbf{\text{Var}}(X)=\displaystyle\sum_{i=1}^n\displaystyle\sum_{j=1}^nw_iw_j\mathbf{\text{Cov}}(X_i,X_j)=\mathbf{w}^\top \Sigma \mathbf{w}$$ where $\Sigma$ is the covariance matrix

Formulation 1. Maximizing Expected Return for a Given Level of Risk¶

  • Maximize the portfolio's expected return while keeping its variance ($\sigma^2_{\text{target}}$) below a specified level.
  • This approach finds the highest-return portfolio for each level of variance on the efficient frontier.

Problem Formulation¶

$$\max_{\mathbf{w}} \quad \mathbf{w}^\top \mathbf{\mu}$$ Subject to:

$$\mathbf{w}^\top \Sigma \mathbf{w} \leq \sigma^2_{\text{target}}$$

$$\mathbf{w}^\top \mathbf{1} = 1$$

Formulation 2. Minimizing Portfolio Variance for a Given Expected Return¶

  • Minimize the portfolio's variance ($\sigma^2_{\text{target}}$) while achieving return $R_{target}$.

  • Find the minimum-risk portfolio for each level of expected return on the efficient frontier.

Problem Formulation¶

$$\min_{\mathbf{w}} \quad \mathbf{w}^\top \Sigma \mathbf{w}$$

Subject to: $$ \mathbf{w}^\top \mathbf{\mu} = R_{\text{target}} $$ $$ \mathbf{w}^\top \mathbf{1} = 1 $$ where $\mathbf{1}$ is a vector of ones, ensuring that the weights sum to 1 (fully invested portfolio).

Efficient Frontier¶

We will use formulation #2 and solve for every $R_{\text{target}}$ which will give us the Efficient Frontier

  • Each portfolio on the efficient frontier is efficient, offering the best possible return for its risk level or the least possible risk for its return level.

Adding Risk-Free Asset: Market Portfolio and Capital Market Line¶

Sharpe Ratio:

  • Measures risk-adjusted return: $\frac{\mu - r}{\sigma}$, where a higher ratio indicates a better return per unit of risk.

Market Portfolio:

  • Represents the optimal diversified portfolio, located on the efficient frontier, providing the highest Sharpe ratio among risky assets.
  • Contains all assets in the market, weighted by market value, offering the best risk-adjusted return by efficiently distributing risk across diverse assets.

Capital Market Line (CML):

  • Extends from the risk-free rate to the market portfolio, representing the best possible risk-return combinations.
  • By adjusting the mix of the risk-free asset and the market portfolio, investors can control their risk exposure and expected return, following a linear risk-return trade-off.
  • Serves as a benchmark for efficient investing, illustrating the optimal balance between risk and return.

Step 1a: Compute Mean Returns or expected return for each stock:

Step 3. Optimization Part 1: Find Efficient Frontier:¶

  • The goal of portfolio optimization is to minimize risk (standard deviation) for a given level of return (or maximize returns for a given level of risk).
  • We use scipy’s minimize function to find the portfolio weights for each Return Target level.
Step 3a. Define Objective Function¶
  • Minimize Portfolio Risk
In [33]:
def portfolio_risk(weights,cov_matrix):
    return np.sqrt(np.dot(np.dot(weights.T,cov_matrix),weights))
def objective_function(weights):
    return portfolio_risk(weights,cov_matrix)
Step 3b. Define Constraints For Each Return Target¶
  • Portfolio Return = Return Target
  • Sum of Weights = 1
In [39]:
def portfolio_return(weights,mean_returns):
    return np.sum(weights*mean_returns)
def constraint_sum_to_1(weights):
    return np.sum(weights)-1

def constraint_return_eq_target_return(weights,target_return):
    return portfolio_return(weights,mean_returns)-target_return
    
   
constraints = [
    {'type': 'eq', 'fun': constraint_sum_to_1},  # Sum of weights = 1
    {'type': 'eq', 'fun': lambda w: constraint_return_eq_target_return(w, return_target)}  # Portfolio return = target return
]

  • Constraints ensure that the sum of the portfolio weights equals 1 (so all capital is invested), and weights remain within bounds (0 to 1,i.e. no short selling).

Step 4a. Market Portfolio:¶
  • Maximize the Sharpe Ratio: This ratio measures the excess return relative to the risk (volatility).
  • In the code, we minimize the negative Sharpe ratio because optimizers minimize by default.
In [58]:
# Define the function to minimize (negative Sharpe ratio)
def neg_sharpe_ratio(weights, risk_free_rate):
    portfolio_ann_return = portfolio_return(weights,mean_returns)
    portfolio_ann_volatility = portfolio_risk(weights,cov_matrix)
    sharpe_ratio = (portfolio_ann_return - risk_free_rate) / portfolio_ann_volatility
    return -sharpe_ratio  # Minimize the negative Sharpe ratio


  
constraints = [
    {'type': 'eq', 'fun': constraint_sum_to_1},  # Sum of weights = 1
   # {'type': 'eq', 'fun': lambda w: constraint_return_eq_target_return(w, return_target)}  # Portfolio return = target return
]


result = optimize.minimize(fun=neg_sharpe_ratio,x0=initial_guess,args=(risk_free_rate,),constraints=constraints,bounds=bounds)
